# Vue-Playground Boilerplate

## Features

- Tailwindcss
- Autoprefixer
- Fontastic Font Icons
  - Heroicons By Steve Schoger. (see icons-reference in public)
  - Social Icons
    - `.icon-twitter-alt`
    - `.icon-youtube-atl`
    - `.icon-github-alt`
    - `.icon-reddit`
    - `.icon-whatsapp`
    - `.icon-mobileme`
    - `.icon-sharethis`
    - `.icon-facebook`
- Prettier

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
